## How to Use Templates

1. Open the blog writer at `/blog-writer`
2. Select a quick template from the dropdown
3. Customize the content with your examples
4. Export as JSON or Markdown
5. Add to your blog directory

## SEO Best Practices

- **Title:** 50-60 characters, include year (2025/2026)
- **Meta Description:** 150-160 characters
- **Keywords:** 5-10 relevant keywords
- **Content:** 800-2000 words minimum
- **Structure:** Use H2, H3 headings, bullet points, examples
- **Internal Links:** Link to practice pages and other articles
